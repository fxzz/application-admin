## 프로젝트 목적

application 프로젝트의 어드민 페이지가 웹 사이트 내에서 구현되어 있습니다.
<br>
관리자 계정의 보안 문제를 방지하기 위해, Nexacro를 활용하여 사이트와 독립적인 어드민을 구축



## 개발 기간

2023.11.05 ~ 2023.11.05 게시글 관리 <br>
2023.11.26 ~ 2023.11.27 댓글 관리, 공지 관리- 등록, 삭제, 수정


## 주요 기능

게시글 관리 - 최신, 신고 순서로 선택 조회, 삭제
